# Brandingdong_iOS

## 개요.
여성 의류 쇼핑몰 BRANDI의 iOS App Clone 프로젝트 입니다.

## 팀원.

- 이진욱 [[gitHub]](https://github.com/jwlee07)
- 윤병일 [[gitHub]](https://github.com/ByoungilYoun)

## use Library

```
- SnapKit
- SwiftLint
- Alamofire
```

## Architecture

```
- MVC
```
## Requirements

```
- Language
    - Swift 5.0

- FramWork
    - UIKit
    - CocoaPods

- IDE
    - Xcode
```

## Project

- 프로젝트 진행과정에 대한 설명입니다.

### 기획안

```
- MVP LIST
  - 로그인
  - 메인 홈 카테고리 (쇼핑몰마켓, 브랜드, 뷰티)
  - 상품상세정보
  - 상품구매
  - 장바구니
  - 찜
  - 마이페이지
```
